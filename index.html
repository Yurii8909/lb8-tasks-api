Tasks API - Лабораторна робота 8

Документація REST API для управління задачами

Посилання:

GitHub репозиторій: https://github.com/Yurii8909/lb8-tasks-api

Сторінка додатку: https://yurii8909.github.io/lb8-tasks-api/

Ендпоінти API:

  • POST /api/tasks - Створити задачу
  • GET /api/tasks - Отримати всі задачі
  • GET /api/tasks/:id - Отримати задачу по ID
  • PUT /api/tasks/:id - Оновити задачу
  • DELETE /api/tasks/:id - Видалити задачу
  • GET /api/tasks/summary - Статистика (агрегація)

Виконані вимоги:

  1. Підключення до MongoDB Atlas
  2. Схема Task з валідацією
  3. Всі CRUD операції
  4. Агрегаційний запит getTasksSummary
  5. GitHub репозиторій з кодом
  6. Сторінка на GitHub Pages
