Tasks API - Лабораторна робота 8

REST API для управління задачами з MongoDB Atlas

Посилання:

  • GitHub репозиторій: https://github.com/Yurii8909/lb8-tasks-api
  • API документація: https://yurii8909.github.io/lb8-tasks-api/

Доступні ендпоінти:

POST /api/tasks - Створити задачу
GET /api/tasks - Отримати всі задачі
GET /api/tasks/:id - Отримати задачу по ID
PUT /api/tasks/:id - Оновити задачу
DELETE /api/tasks/:id - Видалити задачу
GET /api/tasks/summary - Статистика (агрегація)

Вимоги до звіту:

  1. Посилання на GitHub з кодом
  2. Посилання на сторінку з додатком (ця сторінка)
  3. Скріншоти виконання CRUD операцій
  4. Скріншот результату getTasksSummary()
